# 🤖 AI Data Analyst Assistant

An AI-powered data analytics application built using Streamlit and Google Gemini AI.

## Features

* Upload CSV datasets
* Interactive dataset preview
* Dashboard metrics
* Dataset health analysis
* Data visualization
* AI-powered question answering
* AI-generated insights
* Download dataset summary reports

## Technologies Used

* Python
* Streamlit
* Pandas
* Plotly
* Google Gemini AI

## Installation

1. Clone the repository

2. Install dependencies

pip install -r requirements.txt

3. Run the application

streamlit run app.py

## How It Works

1. Upload a CSV file
2. Explore dataset statistics
3. Visualize data
4. Ask questions about the dataset
5. Generate AI-powered insights
6. Download the summary report

## Future Improvements

* PDF report generation
* Advanced visualizations
